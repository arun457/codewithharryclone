Moving Lamborghini Car Animation Using Pure HTML, CSS & JavaScript

Hello guys, in this blog we will make a moving car animation using pure HTML, CSS, and JavaScript. The images of the car, wheels, and background along with the audio are available at the end of the blog which you may download later.

Let's open VS code in the folder. We will make 3 files, named index.html(HTML file, to write HTML), style.css(CSS file, where we will do the styling), and script.js(JavaScript file, in this file we will keep scripting of frontend).

In the index.html file, we will create a boilerplate. If you are using a text editor and there is no emmet abbreviation then you can type it there, and for your time convenience there is one shortcut in VS code(visual studio code) that is '!+Enter'. This will help you to create a boilerplate automatically.

Code Index.html as described/written in the video :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

We have to link a CSS file and JavaScript file in index.html as follows:

<link rel="stylesheet" href="style.css">
<script src="script.js"></script>

Let's Create First Sky

We start with constructing a class of sky inside a class container. Give a relevant title to the website. 

Code Index.html as described/written in the video :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Car Animation CSS and JavaScript</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js"></script>
</head>
<body>
    <div class="container">
        <div class="sky">
            
        </div>
    </div>
</body>
</html>

Now for styling in CSS file (style.css), firstly put a CSS reset. Now, we will target the class sky, to give CSS properties like height, width, and background image as shown in the image given below.

Code style.css as described/written in the video :

*{
    margin: 0;
    padding: 0;
}
.sky{
    height: 100vh;
    width: 100%;
    background-image: url(background.jpg);
    background-repeat: no-repeat;
    position: absolute;
}

If you don't have a  Live Server, an extension of VS code, watch the video attached below and learn how to install and run a Live Server.

Adding Trees, Track, Car

Now to add more elements we will construct three more classes which will add the trees, a track, and a car. Observe the code mentioned below.Code Index.html as described/written in the video :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Car Animation CSS and JavaScript</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js"></script>
</head>
<body>
    <div class="container">
        <div class="sky">
            <div class="trees"></div>
            <div class="track"></div>
            <div class="car"></div>
        </div>
    </div>
</body>
</html>

Individual classes can be styled by adding images, positioning, and numbers.  

Code style.css as described/written in the video :

.trees{
    height: 100vh;
    width: 100%;
    background-image: url(trees.png);
    background-size: cover; 
    position: absolute;
    top: -144px; 
}
 
.track{
    height: 60vh;
    width: 800vw;
    background-image: url(track.png);
    background-repeat: repeat-x; 
    position: absolute;
    top: 70vh;
}
 
.car{
    height: 100px;
    width: 380px;
    background-image: url(car_body.png);
    background-size: cover;
    background-repeat: no-repeat;
    position: absolute;
    left: 444px;
    bottom:30vh;
}

Adding Wheels To The Car

If you observe carefully, the car has no wheels. Hence we need to add wheels to this car by adding a new class named wheel. We will also construct two more classes named wheel1 and wheel2 for the identification of the left wheel and right wheel.

Code Index.html as described/written in the video :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Car Animation CSS and JavaScript</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js"></script>
</head>
<body>
    <div class="container">
        <div class="sky">
            <div class="trees"></div>
            <div class="track"></div>
            <div class="car">
                <div class="wheel wheel1">
                    <img src="car_wheel_left.png" alt="">
                </div>
                <div class="wheel wheel2">
                    <img src="car_wheel_right.png" alt="">
                </div>
            </div>
        </div>
    </div>
</body>
</html>

Now for styling, the wheels add images for individual wheel classes img.

Code style.css as described/written in the video :

.wheel1 img{
    width: 77px;
    position: relative;
    top:42px;
    left:42px;
}
.wheel2 img{
    width: 77px;
    position: relative;
    top: -39px;
    left: 235px;
}

Let’s begin with the animations

  • Animation for wheel rotation

CSS Syntax of @keyframes :

@keyframes animationname {keyframes-selector {css-styles;}}

To create animation for the rotation of the wheels we will use the transform property of animation. The animation-name could be anything, for convenience we will name it as wheelRotation. 

Code style.css as described/written in the video :

@keyframes wheelRotation
{
         100% {
                  transform: rotate(360deg);
         }
}

CSS Syntax of add animation :

animation: name duration timing-function delay  iteration-count direction fill-mode play-state;

The animation must be applied to both the wheels img as follows:

Code style.css as described/written in the video :

.wheel1 img{
    width: 77px;
    position: relative;
    top:42px;
    left:42px;
    animation: wheelRotation linear .16s infinite;
}
 
 
.wheel2 img{
    width: 77px;
    position: relative;
    top: -39px;
    left: 235px;
    animation: wheelRotation linear .16s infinite;
}
  • Animation for movement of track in reverse direction 

This animation will create a visual of the car moving forward while the wheels rotate. The animation-name will be carMove and the transform will be translateX (-500vw). 

Code style.css as described/written in the video :

@keyframes carMove
{
    100%{
        transform: translateX(-500vw);
    }
}

The animation should be applied to the track class as follows.

Code style.css as described/written in the video :

.track{
    height: 60vh;
    width: 800vw;
    background-image: url(track.png);
    background-repeat: repeat-x; 
    position: absolute;
    top: 70vh;
    animation: carMove linear 13s infinite;
}
  • Animation to shake the car

This animation will help to give a look and feel of a moving car. Three keyframes will be required for this one for zero percent, one for 50 percent, and one for 100 percent of animation.

Code style.css as described/written in the video :

@keyframes shake
{
    0%{
        transform: translateY(-5px);
    }
    50%{
        transform: translateY(5px);
    }
    100%{
        transform: translateY(-5px);
    }
}

The animation should be applied to the car class.

Code style.css as described/written in the video :

.car{
    height: 100px;
    width: 380px;
    background-image: url(car_body.png);
    background-size: cover;
    background-repeat: no-repeat;
    position: absolute;
    left: 444px;
    bottom:30vh;
    animation: shake linear 3s infinite;
}

If you scroll towards the right side of the website you will notice the road is overflowing. To resolve this problem you need to mention just one line of code.

Code style.css as described/written in the video :

body{
    overflow: hidden;
}
  • Addition of sound to the car

To add sound to the website we will write some JavaScript code. We will create an element named audio and set its attribute. Now to set the sound on the loop we will type code 'audio.loop=true;' in the javascript file. And to play sound will type 'audio. play;'.

Code script.js as described/written in the video :

var audio = document.createElement('audio');
audio.setAttribute('src', 'sound.mp3'); 
audio.loop=true;
audio.play();
  • Final touch to our website

To beautify the website more we can add shake the body animation.

This is similar to shake car animation, the only difference is in the values that we add. For this we will translate from 50px at 10% to -50px at 50% and then again 50px at 100%.

Code Style.css as described/written in the video :

@keyframes shakebody
{
    0%{
        transform: translateY(-50px);
    }
    50%{
        transform: translateY(50px);
    }    
    100%{
        transform: translateY(-50px);
    }
}

Applying this animation in the body to shake the body.

Code Style.css as described/written in the video :

body{
    overflow: hidden;
    animation: shakebody linear 6s infinite;
}
