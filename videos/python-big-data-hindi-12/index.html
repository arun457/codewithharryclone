[Hindi] Merging DataFrames In Pandas - Python Data Science and Big Data Tutorials

We are clear with the Pandas library, and with the concept of DataFrames. In this section, we will discuss another important topic. Till knowWe know to create DataFrames and perform many operations on the DataFrame with the help of built-in functions.

Here is another operation that can be performed on Dataframes. For doing this we again use the function provided by python.

Let us understand by learning and executing it:

Given below are the steps:

 

Step 1: Open your Windows PowerShell. Type in the below command to start again with Jupyter notebook.

jupyter notebook

Step 2: Import the NumPy module by the below command.

import numpy as np

Step 3: Import pandas module by the below command.

import pandas as pd

Here we are importing both the modules so that it is easier for us to use the functions associated with the library.

Step 4: Loading values in DataFrame. Given below is a very simple command in order to create a DataFrame.

df1 = pd.DataFrame([[1,2,3],[5,7,7],[5,3,7]])

Step 5: To add the header to the DataFrame just use the columns keyword and add the header of our choice. Below is the quick code for understanding.

df1 = pd.DataFrame([[1,2,3],[5,7,7],[5,3,7]],
columns=['A','B','C'])

Step 6: Since we are performing a merge operation, we have to create another DataFrame.

df2 = pd.DataFrame([[11,2,31],[5,7,7],[13,3,33]],
columns=['A1','B1','C1'])

Given below image will help you in better understanding:

 

Step  7: Now to perform the merge operation we can make use of a built-in function called merge()

The quick code is given below

df3=pd.merge(df1,df2,right_on='B1',left_on='B')

As told regarding the documentation, we can know more about this pd.merge function by browsing about it.
