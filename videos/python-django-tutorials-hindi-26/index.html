With this, we have successfully created the carousel for the home page of the shop app. Now, let's add images to the cards. Follow the steps given below :

Step 1: Search "Product Images" on Google Images.

Step 2: Download any image of your choice and save that image as "test.jpg" in the static folder of the shop app.

Step 3: After saving the image, open the index.html file and press Ctrl+R.

Step 4: Now, replace src="..." with src=' {% static "shop/test.jpg" %}'

Step 5: Load static by typing the below command in <body> of index.html file :

{% load static %}

Step 6: Restart the development server.

If you have followed all the steps correctly then you will see a screen like this:
