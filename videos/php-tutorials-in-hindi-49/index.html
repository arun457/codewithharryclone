PHP Forum Project: Adding Login Modals, About & Contact Pages | PHP Tutorial #49

Introduction

In this tutorial, we will add Login Modals, About & Contact Pages for our forum website. So, let's fire up our favorite code editor and start coding.

Adding Jumbotron:

Let's start by adding jumbotron to our main page. Let's head over to the bootstrap components page and use their jumbotron code in our index.php file. You will notice after pasting the code that the jumbotron is not working as it should be. It's because the slider does not have any pictures in it. So, we will head over to Unsplash to get some good pictures and paste the URL in the jumbotron's image source. You will see that the slider is working, but it's a bit big for our screen, so we will change the pic resolution by changing the jumbotron code. Usually, it's 1600x900, and we will use 2400x700(you can always try different values). Now our Carousel homepage is ready. 

Adding the other pages:

Now, it is time to add some new pages, like about.php, contact.php. We will create these pages, and we will only include the usual bootstrap template with the header and footer file. Now in the _header.php file, we will customize the navbar.php file. We will add links to the about page and contact pages. Now we have to create the login modal and signup modal for our website. So, in the partials folder, we will create loginmodal.php and signupmodal.php. Now we will add the modal code in those two files. The modal code can be found on the bootstrap components website. Now, as we have added the modal code, we will include the Loginmodal.php file and Signupmodal.php inside the header using PHP include method. Now, all we have to do is customize the modal files. We need to add forms inside the modal files. For that, we will be using bootstrap forms. We will copy and paste the form code from the bootstrap website. You can customize it according to your needs. 

The Source code is given down below for the modal files. Also, for better understanding, watch the complete video.
