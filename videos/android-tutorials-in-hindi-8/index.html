Creating a Navigation Drawer App with Hamburger Icon In Android | Android Tutorials in Hindi #8

In this video we will learn how to make a navigation drawer app with hamburger icon.

Previously when Android Studio was not there and android apps were made using Eclipse than android developers faced a lot of problems and making navigation drawer apps was hectic. But now we can easily make such apps, we have to select navigation drawer activity in the Android studio . Selecting this activity will make your work much easier as it has a lot of code already written for you.

Fragment is basically a subactivity i.e activity withon another activity.A fragment has its own set of layouts and has its own behavior and its own lifecycle and callbacks.

In nav_header_main.xml file we can change the text of the content from “Android Studio” to “Harry”

You can change the colors of navigation bar as per your choice by going into color.xml file. You can manually select any shade available by just clicking on that shade.

In dimens.xml file you can change the dimensions of the title bar.

To change the icon on the navigation drawer go to the drawable folder do right click and than go to the vector asset and click on the clip art and choose the image of your choice.

After installing the app in your smartphone the icon will be shown as follows:

To change the color of the side navigation bar go to the side_nav_bar.xml in the layout folder and you can change the colors manually according to choice.

To change the icon on the side navigation drawer go to the drawable folder and do right click and than select New > Image Asset  and than you can select the icon of your choice or you can also download the icon of your choice and use it.
