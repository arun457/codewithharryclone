[Hindi] Adding New Posts To Flask Blog - Web Development Using Flask and Python #16

In this blog we will make “edit” page through which we can edit or add new blogs.

First we have to make “edit” endpoint which also means we have to make new webpage, say “edit.html”. We are focusing on backend here so let’s make end point!

Note: In HTML form, put action = “/edit/{{post.sno}}”

@app.route("/edit/<string:sno>" , methods=['GET', 'POST'])

We used “string:sno” here because we don’t want to hardcode page number for every blog. We have done it earlier, click here to check it out!

Then we will simply make a function named “edit” and pass sno as an argument. In that function we will first check if the user is logged in. We learnt how to do it in previous blog. We used the following if statement to check it:

if "user" in session and session['user']==params['admin_user']:

then we will get all the data from form placed in edit page. Strategy is if sno is 0 then add all form data as new post in the database but if sno is any other number then edit that number of blog, simple!

We have fetched data from form alot of times so it shouldn’t be a problem. But here is the code:

if request.method=="POST":
    box_title = request.form.get('title')
    tline = request.form.get('tline')
    slug = request.form.get('slug')
    content = request.form.get('content')
    img_file = request.form.get('img_file')
    date = datetime.now()

Now, we will check if sno is 0 and if it is 0 then we want it to add all the form data in the database. To add data in database we will use “Posts” class which we made earlier, click to check it out! We will pass form data through arguments and then add the data in database through “db.session.add()”. We have already done this earlier, click here to learn about it in detail! Code to do it is:

if sno=='0':
    post = Posts(title=box_title, slug=slug, content=content, tagline=tline, img_file=img_file, date=date)
    db.session.add(post)
    db.session.commit()

Now we will write code if sno is not equal to 0, which means “else”.

In else block we want to edit posts so if we open “/edit/1”, it should open first post, means it should already write all the data in form and we just have to edit in form and submit!

For this, first we will get(filter) the post by sno number and then overwrite all the data of that post with the new edited data! For this code is:

else:
    post = Posts.query.filter_by(sno=sno).first()
    post.box_title = box_title
    post.tline = tline
    post.slug = slug
    post.content = content
    post.img_file = img_file
    post.date = date
    db.session.commit()
    return redirect('/edit/'+sno)

Note: Import redirect from flask

In this we are changing all the values with new values and then redirecting. After all this editing, we will again fetch post(to get the edited data) like this:

post = Posts.query.filter_by(sno=sno).first()

This new edited data will be sent to HTML, where we will put all this new edited data in HTML form through “value” argument.

After all this, the whole “edit” function would look like:

@app.route("/edit/<string:sno>" , methods=['GET', 'POST'])
def edit():
    if "user" in session and session['user']==params['admin_user']:
        if request.method=="POST":
            box_title = request.form.get('title')
            tline = request.form.get('tline')
            slug = request.form.get('slug')
            content = request.form.get('content')
            img_file = request.form.get('img_file')
            date = datetime.now()
        
        if sno=='0':
            post = Posts(title=box_title, slug=slug, content=content, tagline=tline, img_file=img_file, date=date)
            db.session.add(post)
            db.session.commit()
        else:
            post = Posts.query.filter_by(sno=sno).first()
            post.box_title = box_title
            post.tline = tline
            post.slug = slug
            post.content = content
            post.img_file = img_file
            post.date = date
            db.session.commit()
            return redirect('/edit/'+sno)

    post = Posts.query.filter_by(sno=sno).first()
    return render_template('edit.html', params=params, post=post)
