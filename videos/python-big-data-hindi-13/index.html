[Hindi] Working With JSON In Python - Python Data Science and Big Data Tutorials

In this tutorial, we will learn about JSON in python. JSON (JavaScript Object Notation) is a built-in module in python that can be installed or used just by making use of an import statement; here, we don’t have to use PIP.

JSON in Python:

Python has a built-in package called JSON. In Python, JSON exists as a string. Before getting started with this, let us learn about some points which must be kept in mind while working with JSON files.

  • In JSON files, always youse double-quotes.
  • We must never include comments in the JSON file.

Importing JSON module can be done by the below command:

import json

Parse JSON in Python: 

If you have a JSON string, you can parse it by using the json.loads() method. Now it is important for us to create a JSON string.

Here is the quick code for you to create a JSON string:

json_data =  '{ "A":1, "B":2, "C":3}'

Now to parse this string, we can use json.loads() method.

x=json.loads(json_data)
