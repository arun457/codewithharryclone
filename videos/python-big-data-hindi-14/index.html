[Hindi] Python Matplotlib Tutorial - Python Data Science and Big Data Tutorials

In previous videos, we learned about the NumPy module, the Pandas module, and JSON. In this section, we will study about Matplotlib library.

Matplotlib is a cross-platform data visualization and graphical plotting library for Python and its numerical extension NumPy.

Installation:

In Python, everything is made easier for you; not just saying can even prove this.

Step 1: Open your Windows PowerShell. Type in the below command to start the jupyter notebook again.

jupyter notebook

Step 2: Your computer screen will look as shown below:

Click on the New button (upper right), and it will open up a list of choices. In the choice given, choose Terminal.

Step 3: Pip can help us with the installation. Just run the following command:

pip install matplotlib

Importing matplotlib:

We can use the import keywords to use matlotlib. Given below is the code :

from matplotlib import pyplot as plt

matplotlib.pyplot is a collection of functions that make matplotlib work.

We are not using direct import because then all the functions will be imported, and the working and speed of the module will be affected.

Basic plots in Matplotlib :

Given below is the set of code that will help us to plot a graph.

%matplotlib inline 
from matplotlib import pyplot as plt
plt.plot([1,2,3],[2,5,7])
plt.show()

The below image will help you in better understanding:

We even have a function through which we can provide labels to X and Y coordinates.

plt.ylabel(“this is Y axis”)
plt.xlabel(“this is X axis”)

If you want to add the title to the graph, here is the function to do this:

plt.title(“Harry’s data”)

Sometimes we have two data sets given to us, and we would want to plot the data inside the same graph; then, we can use the following code:

# importing matplotlib module 
%matplotlib inline 
from matplotlib import pyplot as plt
# dataset 1 
# x-axis values
x = [1,5,4]
  
# Y-axis values
y = [23,45,89]

# dataset 2 
# x-axis values
X1 = [3,7,3]
  
# Y-axis values
Y1 = [5,8,9]

# Function to plot
plt.plot(x,y)
plt.plot(X1,Y1)
# function to show the plot
plt.show()
