Embedding Video with media controls in Android | Android Tutorials in Hindi #9

In this video we will learn how to embed a video in our android app.

In our journey to become an android developer it is obvious to make an app which plays a video.

Lets  start a new project with Empty Activity, and  name  it as  “HarryTube”.

I have used a video  from my python series in which I had  made a  DRS system. You can use any video of your choice.

Now go to app> res and do right click and than click on New > Android Resource Directory and than name that directory as “raw”  resource type as “raw” and than click “ok”. Now copy and paste your video into the “raw” folder.

Delete the TextView “Hello World ” which is already present in your app and drag and a VideoView and do the constaraints accordingly. Set the width of the VideoView (layout_width attribute ) as “match_constarint ” . Try to design the whole layout as youtube or as per your requirement.

Now drag and drop a TextView and do the constaraints and place it accordingly and change its text from “TextView” to “Comments ”.

Set the text size of the text view as 30sp and its width and height as “wrap_content ”.

 Search the “id” attribute of VideoView  and  set  the id as “cricket”.

Drag and drop another TextView  and set its position next to the  already existing TextView and write any number within this TextView  and wite any random number inside that TextView, it will appear as  if it is showing the count of the comments on that video.

 

To play the video which is being copied to the “raw” directory you have to write the following code:

VideoView cricket = findViewById(R.id.cricket);

        cricket.setVideoPath("android.resource://" + getPackageName() + "/" + R.raw.clip);

        cricket.start();

To make a video controller include the following lines in your code before cricket.start() line :

        MediaController mediaController = new MediaController(this);

        cricket.setMediaController(mediaController);

        mediaController.setAnchorView(cricket);

After writing the code run the project and wait until it gets installed in your emulator .

Code as described/written in the video

package com.codewithharry.harrytube;

import androidx.appcompat.app.AppCompatActivity;

import android.os.Bundle;
import android.widget.MediaController;
import android.widget.VideoView;

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        VideoView cricket = findViewById(R.id.cricket);
        cricket.setVideoPath("android.resource://" + getPackageName() + "/" + R.raw.clip);
        MediaController mediaController = new MediaController(this);
        cricket.setMediaController(mediaController);
        mediaController.setAnchorView(cricket);
        cricket.start();
    }
}
