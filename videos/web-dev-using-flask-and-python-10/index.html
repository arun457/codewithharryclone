[Hindi] Making Parameters Configurable In Flask - Web Development Using Flask and Python #10

In this blog we will make our flask application configurable. What it means is, if some person wants to use our flask application then they don’t have to go through the whole code, they can just open ‘config.json’ file and make changes in that which will reflect in the whole website automatically!

Why use config.json? How does this work?

I will try to explain it with the help of an example. I will show you two codes which will help you understand why use this method and how does it work? After seeing these codes you tell which one is better way of writing the same thing on the basis of simplicity and configurability.

First:

print("CodeWithHarry")
print("Channel's name is CodeWithHarry")
print("CodeWithHarry has flask tutorials")
print("Subscribe CodeWithHarry!")

Second:

channel_name = "CodeWithHarry"
print(channel_name)
print(f"Channel's name is {channel_name}")
print(f"{channel_name} has flask tutorials")
print(f"Subscribe {channel_name}!")

Ofcourse second one is better, right? We just have to change “channel_name” variable and it will reflect in all lines. Until now we were using first method, hard coding everything but from now on we will use second method. We will make a dictionary in which there will be list of items which will act like variables, like there could be variable “websitename”, anybody can just change value of that variable and the whole website would change!

Enough theory, now let’s do some coding!

We will use json module for this so import json in your python file. Make “config.json”. You can make a variable “params” in json file which will have all our parameters. Code is like this:

{
    "params": {
        "websitename": "CodingThunder"
    }
}

Taking variables from JSON file in flask app:

import json
with open('config.json', 'r') as c:
    params = json.load(c)["params"]

Here we opened json file in read mode and took “params” variable(from json file) and put it equal to another “params” variable(in python file) from which we can take all other variables! Now we have a separate params variable in python which has all the list items of “params” from json file. Now it’s a piece of cake. We can just write params[‘variable_name’] and get the value. Like in the example given above we made a variable “websitename” in json file, if we want that variable we can just write:

params['websitename']

We can make more variables and get them like this.

Now with this way we can use json variables in our python file/flask app but what about our website? We want these variable in HTML too. So let’s learn how to do that!

How to use JSON variables in HTML file?

Sending JSON to HTML:

First we have to pass those variables to HTML file like this:

@app.route("/")
def home():
    return render_template('index.html', params=params)

We passed the whole “params” to index.html page. Like this we can pass it to other pages too.

Using JSON variables in HTML:

Now, we have params in HTML but how to use it? Like in python we used it like params['websitename'] but we can’t do the same in HTML. So, how to do this in HTML?

Answer is jinja! Like we had ‘websitename’ variable, if we want to put it in heading tag. We can write it like:

<h1>{{params['websitename']}}</h1>

Similarly, we can use any variable from params in any HTML tag like this. You can go in json file and change any variable’s value and it will change everywhere automatically.

1. main.py file as described in the video

from flask import Flask, render_template, request
from flask_sqlalchemy import SQLAlchemy
from flask_mail import Mail
import json
from datetime import datetime


with open('config.json', 'r') as c:
    params = json.load(c)["params"]

local_server = True
app = Flask(__name__)
app.config.update(
    MAIL_SERVER = 'smtp.gmail.com',
    MAIL_PORT = '465',
    MAIL_USE_SSL = True,
    MAIL_USERNAME = params['gmail-user'],
    MAIL_PASSWORD=  params['gmail-password']
)
mail = Mail(app)
if(local_server):
    app.config['SQLALCHEMY_DATABASE_URI'] = params['local_uri']
else:
    app.config['SQLALCHEMY_DATABASE_URI'] = params['prod_uri']

db = SQLAlchemy(app)


class Contacts(db.Model):
    sno = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    phone_num = db.Column(db.String(12), nullable=False)
    msg = db.Column(db.String(120), nullable=False)
    date = db.Column(db.String(12), nullable=True)
    email = db.Column(db.String(20), nullable=False)

@app.route("/")
def home():

    return render_template('index.html', params=params)


@app.route("/about")
def about():
    return render_template('about.html', params=params)


@app.route("/contact", methods = ['GET', 'POST'])
def contact():
    if(request.method=='POST'):
        name = request.form.get('name')
        email = request.form.get('email')
        phone = request.form.get('phone')
        message = request.form.get('message')
        entry = Contacts(name=name, phone_num = phone, msg = message, date= datetime.now(),email = email )
        db.session.add(entry)
        db.session.commit()
        mail.send_message('New message from ' + name,
                          sender=email,
                          recipients = [params['gmail-user']],
                          body = message + "\n" + phone
                          )
    return render_template('contact.html', params=params)


app.run(debug=True)
